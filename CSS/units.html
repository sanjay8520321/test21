Pixel - Absolute unit.


cm - centimeters


Percent - Relative to the parent element.


em - Relative to the size of the parent element (2em means 2 times the size of the default/current font)

This is 20px - Hellow
This is 2em - Hellow

rem - Relative to size of the root element (2rem means 2 times the size of the root font)

This is 20px - Hellow
This is 2em - Hellow

At first you may not notice any difference in em and rem but when we nest elements, then we can see the difference. To know more click here or check the example below.

em

    MY LIST
  • one
  • two
    • two-a
    • two-b
    • two-c
      • two-c-1
      • two-c-2

rem

    MY LIST
  • one
  • two
    • two-a
    • two-b
    • two-c
      • two-c-1
      • two-c-2

vw and vh - 1% of the viewport's width and height. Viewport is the browser window size. If the viewport is 50cm wide, 1vw = 0.5cm.
